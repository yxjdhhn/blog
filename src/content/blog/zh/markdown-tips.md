## Markdown 基础回顾

Markdown 是一种轻量级标记语言，让你可以用简洁的语法撰写格式化文本。

### 文本格式

- **粗体文本** 使用 `**text**`
- *斜体文本* 使用 `*text*`
- ~~删除线~~ 使用 `~~text~~`
- `行内代码` 使用反引号

### 列表

有序列表：

1. 第一项
2. 第二项
3. 第三项

无序列表：

- 苹果
- 香蕉
- 橙子

### 代码块

使用三个反引号可以创建代码块，并指定语言进行语法高亮：

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

for i in range(10):
    print(fibonacci(i), end=' ')
```
