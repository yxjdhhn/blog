## 欢迎来到元霄节快乐的博客

你好！欢迎来到我的个人博客。这是我的第一篇文章，我想借此机会聊聊为什么要开始写博客。

### 为什么写博客？

写博客对我来说有几个重要的意义：

1. **知识沉淀** — 将学到的知识系统化地记录下来，加深理解
2. **分享交流** — 与更多人分享我的经验和观点
3. **持续学习** — 写作的过程本身就是学习和思考的过程

### 这个博客的技术栈

这个博客使用了以下技术构建：

- **Astro** — 一个以内容为中心的现代静态站点生成器
- **Tailwind CSS** — 原子化 CSS 框架
- **Markdown** — 使用 Markdown 撰写文章

下面是一个简单的代码示例：

```javascript
function greet(name) {
  return `你好，${name}！欢迎来到我的博客。`;
}

console.log(greet('世界'));
```
